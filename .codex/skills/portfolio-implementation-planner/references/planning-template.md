# Planning Template

## Plan Shape

Use this shape:

1. Goal
2. Scope
3. Assumptions
4. Tasks
5. Acceptance criteria
6. Verification
7. Risks
8. Public-safe checks

## Task Rules

- Keep tasks small.
- Group by page, component, or data layer.
- Include verification for each phase.
- Preserve no-implementation mode if the user asks for planning only.

## Required Checks

- responsive behavior
- accessibility
- content safety
- theme consistency
- reduced motion
- contact form if affected
