# Architecture Checklist

## Routing

- `/`
- `/work`
- `/work/client-projects`
- `/work/plugins`
- `/work/innovative-projects`
- `/work/accelerators`
- `/work/[slug]`
- `/expertise`
- `/plugins-and-accelerators`
- `/about`
- `/resume`
- `/contact`
- `/thanks`

## Stack

- Astro.
- React and TypeScript.
- Tailwind CSS.
- Motion for React.
- GitHub Pages hosting.
- GitHub Actions deployment.
- Splitforms form.

## Performance

- Static render content pages.
- Use React only where interactivity is needed.
- Optimize images.
- Lazy-load noncritical visuals.
- Avoid heavy background animation.

## SEO

- Unique metadata per page.
- Open Graph data.
- JSON-LD where useful.
- Sitemap and robots.
