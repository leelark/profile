# Design Review Checklist

## Theme

- Dark theme is default.
- Light theme is fully designed.
- Purple is accent only.
- Contrast is strong.
- Focus ring visible.

## Layout

- No text overflow.
- No nested cards.
- No decorative blobs.
- Mobile layout is single column.
- Filters work on mobile.
- Diagrams collapse cleanly.

## Motion

- Animations are subtle.
- Reduced motion works.
- No scroll-jacking.
- No typewriter hero.
- No bouncing cards.
- Theme switch does not flash.
- Button/link motion completes in 120 to 180 ms.
- Filter/nav motion completes around 180 ms.
- Card/accordion motion completes in 220 to 280 ms.
- Page/hero entrance stays under 420 ms.
- Reduced motion disables scroll reveal, stagger, scale, animated glow, and parallax.

## Content Hierarchy

- Role and value visible in first viewport.
- Featured work appears before long bio.
- Project cards show category and evidence.
- Case studies start with executive summary.
