# QA Checklist

## Responsive Widths

- 320 px
- 375 px
- 390 px
- 430 px
- 768 px
- 1024 px
- 1280 px
- 1440 px

## Accessibility

- skip link
- landmarks
- heading order
- keyboard navigation
- visible focus
- labels and errors
- reduced motion
- contrast

## Performance

- Lighthouse Performance 90+
- Accessibility 95+
- Best Practices 95+
- SEO 95+
- LCP under 2.5 s
- CLS under 0.1
- INP under 200 ms

## Content Safety

- no secrets
- no raw logs
- no private URLs
- no Appian object IDs
- no unverified claims
- confirmation notes hidden from public mode

## SEO and Structured Data

- unique title per page
- unique meta description per page
- canonical URL
- Open Graph metadata
- Person JSON-LD where appropriate
- WebSite JSON-LD on home
- BreadcrumbList on project pages
- sitemap and robots reviewed

## Browser Matrix

- latest Chrome
- latest Edge
- latest Firefox
- latest Safari
- iOS Safari
- Android Chrome
