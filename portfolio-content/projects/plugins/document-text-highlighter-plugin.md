## Hero Summary

Reusable Appian component plugin for highlighting document text or regions in Appian and external PDF/image documents using coordinate input.

## Project Context

The old portfolio lists Document Text Highlighter as a component plugin. The Automobile Traceability page also mentions a JavaScript Appian Document-Viewer plugin capable of highlighting external and Appian PDF/image documents by passing coordinates.

This plugin should be positioned as the foundation or earlier version of the more advanced custom HTML document viewer/document intelligence component.

## Problem

Reviewers often need to identify exact evidence in a document. Without highlighting, users must manually search pages and interpret which content supports a decision.
