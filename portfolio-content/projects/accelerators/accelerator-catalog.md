## Purpose

This catalog lists the Appian Accelerate accelerator workstreams that should be represented in the portfolio. Each item is positioned as a reusable advisory asset, technical pattern, UX pattern, diagnostic approach, integration pattern, or productized solution concept.

The catalog should be used by the future website to power the "Plugins and Accelerators" page, the Appian Accelerate case study, and filtered work views.

## Public-Safe Publishing Rules

Do not publish:

- raw client names unless approved
- environment names
- server names
- private URLs
- raw Appian object IDs
- credential files
- private key files
- exact internal ticket numbers
- raw Health Check rows
- raw logs
- unredacted workbook extracts

Use public-safe language:

- "enterprise client"
- "regulated workflow"
- "self-managed Appian environment"
- "cloud Appian environment"
- "platform health data"
- "source materials reviewed"
- "implementation outcome needs confirmation"
