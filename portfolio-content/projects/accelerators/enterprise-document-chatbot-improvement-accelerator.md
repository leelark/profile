## Hero Summary

Enterprise document chatbot improvement accelerator focused on trust, source formatting, evidence highlighting, admin-configurable guidance, indexing visibility, and unanswered-question handling.

## Project Context

Enterprise knowledge search experiences need more than generated answers. Users need clear source references, formatted supporting text, document highlighting, administrator-controlled guidance, and predictable behavior when the system cannot answer.

## Problem

The reviewed material identified improvement needs around:

- configurable system-level guidance prompts
- introductory disclaimer messages
- formatted supporting text
- source names instead of raw URLs
- evidence highlighting in source documents
- clarification questions
- better unanswered-question behavior
- indexing only useful current documents
- overnight delta indexing
- reporting on indexed content
